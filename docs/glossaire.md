Hyperviseur : logiciel qui permet de créer et gérer plusieurs machines virtuelles sur un même matériel physique.
VM (Machine Virtuelle) : environnement isolé qui simule un ordinateur complet (OS, CPU, RAM) à l’intérieur d’un autre système.
Snapshot : copie instantanée de l’état d’une machine virtuelle à un moment précis, permettant de revenir en arrière.
Vulnérabilité : faiblesse dans un système qui peut être exploitée par un attaquant.
Menace : événement ou acteur potentiel capable d’exploiter une vulnérabilité pour causer un dommage.
Risque = Probabilité × Impact
Impact : gravité des dommages causés lorsqu’un risque se réalise.
CIA Triad : modèle de sécurité basé sur trois piliers : confidentialité, intégrité et disponibilité des données.
Supply Chain Attack : attaque qui cible un fournisseur ou un processus de distribution pour compromettre indirectement les clients.
Ransomware : logiciel malveillant qui chiffre les données d’une victime pour exiger une rançon en échange de leur déchiffrement.
RCE (Remote Code Execution) : capacité pour un attaquant d’exécuter du code à distance sur un système cible.
