# FortressLab — Nexus Corp

Infrastructure fictive d'entreprise construite pour l'apprentissage
de la cybersécurité de zéro à expert.

## Architecture

- **srv-web** : serveur web (DMZ)
- **srv-internal** : serveur interne
- **srv-siem** : supervision et logs
- **win-dc** : Active Directory Windows Server
